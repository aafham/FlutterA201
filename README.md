# FlutterA201

This repository contains 2 Flutter projects:

- `food_ninja`: main app (splash, login, register, remember me)
- `food_ninja_fb`: basic Flutter template/experiment project

## Structure

```text
FlutterA201/
  food_ninja/
  food_ninja_fb/
```

## How To Run

1. Go to one project folder:

```bash
cd food_ninja
```

or

```bash
cd food_ninja_fb
```

2. Install dependencies:

```bash
flutter pub get
```

3. Run the app:

```bash
flutter run
```
