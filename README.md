# FlutterA201

Repository ini berisi 2 project Flutter:

- `food_ninja`: app utama (splash, login, register, remember me)
- `food_ninja_fb`: project Flutter template/eksperimen

## Struktur

```text
FlutterA201/
  food_ninja/
  food_ninja_fb/
```

## Cara Jalanin

1. Masuk ke folder project:

```bash
cd food_ninja
```

atau

```bash
cd food_ninja_fb
```

2. Install dependency:

```bash
flutter pub get
```

3. Run app:

```bash
flutter run
```
